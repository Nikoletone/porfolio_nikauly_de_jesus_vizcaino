R3st0 Modération

Application de bureau Java MVC destinée aux modérateurs et responsables du site R3st0.fr, développée en équipe selon la méthode Agile avec gestion de base de données distante.

Java Swing UI MariaDB Pattern MVC Méthode Agile GitLab

Le Projet

Contexte

Les propriétaires du site r3st0.fr souhaitaient se doter d'une application Java installée sur les postes de travail des modérateurs pour contrôler les avis déposés. L'outil permet de supprimer ou de bloquer la publication des avis ne respectant pas la charte.

Objectifs et Architecture

  • Développement en langage Java en respectant le patron de conception MVC (Modèle-Vue-Contrôleur).
  • Conception d'une interface graphique avec authentification sécurisée et chiffrement.
  • Connexion JDBC à une base de données distante MariaDB mutualisée.

Fonctionnalités développées

  • Authentification : Gestion de différents rôles (modérateur, administrateur, client) stockés en BDD.
  • Listing des avis : Affichage des avis par date décroissante ou par semaine.
  • Modération : Possibilité de masquer un avis avec motif, de le démasquer ou de le supprimer définitivement de la base.

Gestion de Projet Agile

  • Projet découpé en 3 itérations (sprints) avec animation de "Daily Scrums" à chaque séance.
  • Priorisation du Backlog avec la méthode MoSCoW (Must, Should, Could, Won't) et estimation de complexité.
  • Utilisation de GitLab pour la gestion de version et le travail collaboratif.
